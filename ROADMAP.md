# Roadmap (RU)

План на 2024–2025
- Q3 2024: Базовый user-based kNN, CLI, пример данных.
- Q4 2024: Улучшение качества — нормализация рейтингов, топ-N с фильтрами.
- Q1 2025: Метрики (precision/recall, MAP), отчёты.
- Q2 2025: Холодный старт — популярность по категориям, контентные признаки.
- Q3 2025: Инкрементальное обновление матрицы, сохранение артефактов.
- Q3–Q4 2025: Эксперименты: item-based, hybrid, A/B.
