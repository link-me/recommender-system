# Recommender System

Стек: Python + Pandas

Простой демо-проект рекомендательной системы на основе пользовательских оценок. Используется Pandas для подготовки матрицы пользователь×товар и косинусная близость для пользовательского kNN. Есть fallback по популярности.

## Быстрый старт

1) Установить зависимости (рекомендуется виртуальное окружение):

```
python -m venv .venv
.\.venv\Scripts\pip install --upgrade pip
pip install -r requirements.txt
```

2) Запуск с примером данных:

```
python src/main.py --data data/sample_interactions.csv --user u1 --top 5
```

Пример вывода:

```
{"user": "u1", "recommendations": [["i5", 3.5], ["i4", 2.8], ["i6", 1.9]]}
```

3) Fallback по популярности (без указания пользователя):

```
python src/main.py --data data/sample_interactions.csv --fallback --top 5
```

## Формат данных

CSV с колонками: `user_id,item_id,rating`. Если `rating` отсутствует, воспринимается как 1 (имплицитное событие).

## Архитектура

- `src/recommender.py`: ядро — загрузка данных, построение матрицы, расчёт косинусной близости, рекомендации.
- `src/main.py`: CLI-обёртка для локального запуска.
- `data/sample_interactions.csv`: пример датасета.

## Ограничения и развитие

- Алгоритм — простой user-based kNN, без нормализации по средним пользователям и без фильтрации по свежести.
- В `ROADMAP.md` описаны шаги развития (персонализация, метрики качества, холодный старт и т. п.).
